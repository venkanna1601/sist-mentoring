💡 This app works completely free using Google Apps Script + Google Drive as the backend — no servers needed!

Step 1. Open Google Drive → Create a folder named SIST_Mentoring_Records

Step 2. Go to script.google.com → Create new Apps Script project

Step 3. Paste the provided backend script (Apps Script / doPost function)

Step 4. Deploy as Web App → Set access to "Anyone with Google Account"

Step 5. Host this HTML file on GitHub Pages (free) or Netlify (free)
